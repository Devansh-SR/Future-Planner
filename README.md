# 🛡️ Future Planner: Desktop Mission Control
**A professional productivity ecosystem for CSE students and developers.**

![Version](https://img.shields.io/badge/Version-1.0.0-blue)
![Python](https://img.shields.io/badge/Python-3.9+-green)
![Flask](https://img.shields.io/badge/Framework-Flask-emerald)

## 🚀 Key Features
- **Mission Dashboard**: Real-time project health calculated across Academic, Project, and Personal tracks.
- **Interactive Calendar**: High-performance grid with color-coded priorities and categories.
- **Resource Intel**: Integrated storage for methodology, code snippets, and resource links.
- **Time Horizon Tracker**: Dynamic urgency coding and countdowns for all pending deadlines.
- **Visual Progress Sync**: Interactive sliders with milestone celebrations (Confetti) at 100%.
- **Zero-Cloud Architecture**: Local-first data management via `data.json`.
- **Professional Export**: One-click PDF generation for offline monthly documentation.

## 🛠️ Technical Stack
- **Backend**: Python, Flask
- **Frontend**: Tailwind CSS (Optimized for Dark Mode), JavaScript (ES6)
- **Libraries**: FullCalendar, Canvas-Confetti, html2pdf.js

### 🖱️ Quick Launch (Windows)
Simply double-click `launch_planner.bat` to automatically start the server and open the dashboard in your browser.
